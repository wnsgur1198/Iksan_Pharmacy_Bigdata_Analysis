## 주요 기능
* 전북 익산시에서 약국을 개업한다고 전제했을 때, 공공데이터를 사용하여 사업 현황을 조사하고 어디에 개업할지 분석함

    * 수집한 자료 전처리 - 영업상태/읍면제외 등 필터링
